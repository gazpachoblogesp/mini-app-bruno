# Приложение для изучения языков с Бруно

Веб-приложение для изучения испанского языка с маскотом Бруно.

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:5173 в браузере.

## Технологии

- **React 18** + TypeScript
- **Vite** — сборщик
- **Tailwind CSS 3** — стилизация
- **Framer Motion** — анимации
- **Wouter** — роутинг
- **Sonner** — уведомления
- **Lucide React** — иконки

## Структура проекта

```
src/
├── main.tsx           # Точка входа
├── App.tsx            # Роутинг
├── index.css          # Глобальные стили
├── components/ui/     # UI компоненты
├── lib/utils.ts       # Утилиты
└── pages/
    ├── Home.tsx       # Главная страница
    ├── Lesson.tsx     # Интерактивные уроки
    ├── Words.tsx      # Повторение слов
    ├── Practice.tsx   # Практика диалогов
    ├── Profile.tsx    # Профиль пользователя
    └── Path.tsx       # Путь обучения
public/
└── images/            # Изображения маскота
```

## Страницы

| Путь | Описание |
|------|----------|
| `/` | Главная с маскотом и быстрым доступом |
| `/lesson` | Интерактивный урок с тестами |
| `/words` | Повторение слов (флэш-карточки) |
| `/practice` | Практика диалогов с чат-ботом |
| `/profile` | Профиль со статистикой |
| `/path` | Путь обучения с прогрессом |

## Цветовая палитра

- **Primary (оранжевый)**: `hsl(25 95% 55%)`
- **Background (кремовый)**: `hsl(40 33% 96%)`
- **Foreground (коричневый)**: `hsl(30 15% 25%)`

## Шрифт

Google Font **Nunito** — округлый, дружелюбный шрифт.
